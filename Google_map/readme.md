# Google Map Project

This project demonstrates the integration of Google Maps into a web application. It showcases how to embed an interactive map, add markers, and utilize Google Maps API features.

## Features
- Display an interactive Google Map.
- Add custom markers to specific locations.
- Basic usage of the Google Maps API.

## Prerequisites
To use this project, you need:
- A valid Google Maps API key.
- Basic knowledge of HTML, JavaScript, and CSS.

## Important Note
The provided Google Maps API key is **restricted** and will only work on the specified webpage included in this project. **Do not copy-paste it into other projects or webpages**, as it will not function elsewhere. Save time by not attempting to reuse this key in other environments.

## How to Run
1. Clone this repository.
   ```bash
   git clone https://github.com/rishabharaj/basic-projects.git
   ```
2. Navigate to the Google_map folder.
   ```bash
   cd basic-projects/Google_map
    ```

3. Open the index.html file in any browser to view the map.
